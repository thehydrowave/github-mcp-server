# GitHub MCP Server\n\nCustom MCP server deployed on Vercel to connect Claude to GitHub.\n\n## Tools\n\n- **git_read** - Read a file or list a directory\n- **git_push** - Create or update a file\n- **git_delete** - Delete a file\n- **git_list_repos** - List repositories\n\n## Setup\n\nSet `GITHUB_PAT` environment variable in Vercel with a GitHub Personal Access Token.\n\n## MCP Endpoint\n\n`https://<your-domain>/api/mcp`\n
